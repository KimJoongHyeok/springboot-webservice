# springboot-webservice

이동욱(jojoldu)님의 "스프링 부트와 AWS로 혼자 구현하는 웹 서비스" 책을 읽으면서 개발해나가는 레포지토리입니다.
챕터마다 하나씩 구분해나가며 커밋을 진행하였습니다.
Chap 6부터는 AWS 생성과 RDS에 관련한 내용이라 커밋이 없습니다. 그래서 이 내용은 블로그에 글을 정리했습니다.

### 사용 기술
- Java 8
- SpringBoot 2.1.9
- gradle 4.10.2
- Lombok<BR>
- Spring Security
- OAuth 2.0
- JPA<BR>
- JUnit4<br>
- mustache<BR>
- AWS EC2<BR>
- AWS RDS<BR>
- IntelliJ<BR>
- h2 database<BR>
